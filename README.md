# Лабораторная работа 3

Программа для генерации/считывания, сортировки и вывода данных о научных работах.

## Функциональность

- Генерация случайных записей о научных работах
- Сортировка записей по году (возрастание/убывание)
- Вывод данных в форматированном виде
- Чтение/запись из/в файлов

## Использование

## Генерация данных
.\program.exe --generate N
## Чтение данных
.\program.exe --in="input.txt"
## Сортировка данных
.\program.exe --sort --type=asc
.\program.exe --sort --type=desc
## Вывод данных
.\program.exe --out="output.txt"
